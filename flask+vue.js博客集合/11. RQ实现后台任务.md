# RQ实现后台任务
## 1. Redis Queue
RQ (Redis Queue) 是 Python 中用来实现简单的 任务队列，它比 Celery 更加易用，但它的 broker 只支持 Redis，而 Celery 的 broker 支持 RabbitMQ、Redis、Amazon SQS
